## Getting Started

### First time setup

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

### Future runs

```bash
# Activate virtual environment
source venv/bin/activate

# Run the app
streamlit run app.py
```
